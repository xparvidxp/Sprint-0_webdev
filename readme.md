#Sprint 0 - Minor Webdesign

Een website waarop ik mezelf voorstel

Proces:

* Dag 1 - Ik heb een plan gemaakt voor hoe ik de website eruit wil laten zien.
Het idee is een 3D model van het gekozen eendje, waarbij verschillende plekken van het eendje 
informatie op lat komen in tekst vorm. Het eendje is geanimeerd.
Aan het einde van de dag werd duidelijk dat dit te hoog gegrepen is voor mijn niveau op het moment.

* Dag 2 - Gekozen om eerst de minimale eisen van de website te voldoen, en later elementen bij te voegen wanneer ik er aantoe ben. Tijd gespendeerd aan het begrijpen van de workshop API's linken. 
Dit kostte allemaal 3 uur. Ik heb geleerd hoe je API's linkt en heb een begin gemaakt met CSS begrijpen. Ik ga morgen door met het opfrissen van CSS en wellicht ook animeren.
